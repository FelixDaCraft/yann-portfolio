CUPMETRICS.

Anonymiser un concours sans biais — sans que les jurés voient qui produit quoi.

Plateforme événementielle multi-tenant qui gère un concours de A à Z : inscription producteurs, paiements, espace organisateur, espace jury, prix, expositions.

06 · Case study 02

BABYMONITOR.

Quand un protocole propriétaire refuse de parler — on le démonte.

J'ai démonté le protocole propriétaire d'une caméra IP Xiaovv —
V308 sur TCP:8800, chiffrement AES-128-ECB, décodage H.264 / HEVC.
J'en ai fait BabyMonitor, une PWA self-hostée pour surveiller mon bébé en temps réel, avec détection ML embarquée des pleurs (YAMNet TFLite, classification audio AudioSet) et push iOS qui perce le mode silencieux (ntfy en fallback).

07 · Case study 03

RINGDO.

Transcrire chaque appel — automatiquement, depuis Android.

Une app Android native qui surveille en silence le dossier d'enregistrements d'appels et upload chaque nouveau fichier vers un backend self-hosted.
Côté serveur : Mistral en speech-to-text sur l'audio brut, puis le même LLM produit la journalisation structurée de l'appel (date, durée, interlocuteurs, résumé, sujets). Recherche full-text dans l'historique.

09 · Case study 05

JLLM.

Un LLM qui ne peut pas mentir sur un programme politique — parce qu'il ne répond que depuis le texte.

Assistant IA sur le programme l'Avenir en commun : chaque réponse est sourcée, citée (chapitre · section) et vérifiable — 99 % de fidélité au texte, zéro invention.
RAG hybride : BGE-M3 (dense + sparse + ColBERT) → Qdrant (fusion RRF) → reranking ColBERT → génération encadrée par Mistral (souverain, temp. 0). Refus systématique du hors-programme (7/7 aux tests d'injection). Opéré sur mon homelab.
  • Python
  • FastAPI
  • Qdrant
  • BGE-M3
  • Mistral
  • Docker
  • Cloudflare Tunnel
